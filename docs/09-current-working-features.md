### ✅ **정상 동작하는 기능들**

#### **로그인/로그아웃**
- **URL**: `http://localhost:9960/login`
- **기능**:
  - JWT 토큰 기반 인증
  - 역할 기반 접근 제어 (USER, MANAGER, ADMIN)
  - 자동 토큰 갱신
  - 로그아웃 시 토큰 무효화

#### **테스트 계정**
```
ABC기업 계정:
1. 관리자 계정
   - 사용자명: admin
   - 비밀번호: admin123
   - 권한: ADMIN
   - 소속: ABC기업 인사팀

2. 매니저 계정
   - 사용자명: manager
   - 비밀번호: manager123
   - 권한: MANAGER
   - 소속: ABC기업 개발팀

3. 일반 사용자 계정
   - 사용자명: user
   - 비밀번호: user123
   - 권한: USER
   - 소속: ABC기업 개발팀

XYZ그룹 계정:
4. XYZ 관리자
   - 사용자명: xyz_admin
   - 비밀번호: xyz123
   - 권한: ADMIN

5. XYZ 매니저
   - 사용자명: xyz_manager
   - 비밀번호: xyz123
   - 권한: MANAGER

DEF코퍼레이션 계정:
6. DEF 관리자
   - 사용자명: def_admin
   - 비밀번호: def123
   - 권한: ADMIN

7. DEF 사용자
   - 사용자명: def_user
   - 비밀번호: def123
   - 권한: USER
```

#### **보안 기능**
- 비밀번호 BCrypt 암호화
- JWT 토큰 만료 관리
- 보호된 라우트 접근 제어
- CORS 설정 완료

- **데스크톱**: 브라우저에서 PWA 설치 가능

- **완전한 오프라인 지원**:
  - 기본 페이지 오프라인 로딩
  - Google Fonts 캐싱 (1년간)
  - API 데이터 캐싱:
    - 부서/직급: 24시간 캐싱
    - 직원: 2시간 캐싱
    - 일반 API: 24시간 캐싱

#### **캐싱 전략**
- **StaleWhileRevalidate**: 부서/직급 정보 (읽기 전용)
- **NetworkFirst**: 직원/일반 API (실시간성 중요)
- **CacheFirst**: Google Fonts (정적 리소스)

#### **백그라운드 동기화**
- 자동 업데이트
- 캐시 자동 관리
- 오래된 캐시 자동 정리

## 🧪 테스트 방법

### **1. 로그인 테스트**
```
1. http://localhost:9961 접속
2. 다음 계정 중 하나로 로그인 테스트:
   - 관리자 권한: admin/admin123
   - 매니저 권한: manager/manager123
   - 일반 사용자 권한: user/user123
3. 권한별로 접근 가능한 메뉴 확인
4. 대시보드 화면 확인
```

### **2. 인사관리 테스트**
```
1. 로그인 후 "인사관리" 메뉴 클릭
2. 직원 목록 확인
3. "직원 등록" 버튼 클릭
4. 새 직원 정보 입력 및 저장
5. 검색 및 필터링 기능 테스트
```

### **3. 공통 관리 테스트**
```
1. "공통 관리" 메뉴 클릭
2. 회사/부서/직급 관리 화면 확인
3. 새 항목 등록/수정/삭제 테스트
```

### **4. PWA 테스트**
```
1. 모바일 브라우저에서 접속
2. "홈 화면에 추가" 옵션 확인
3. 오프라인 모드에서 기본 기능 동작 확인
```

### **5. API 테스트**
```
1. http://localhost:9961/swagger-ui.html 접속
2. 인증 토큰 설정 (Bearer token)
3. 각 API 엔드포인트 테스트
```

## 📞 문제 발생 시

### **일반적인 문제 해결**
1. **로그인 안됨**: API URL 및 CORS 설정 확인
2. **데이터 안보임**: 백엔드 서버 실행 상태 확인
3. **빌드 오류**: TypeScript 컴파일 오류 확인
4. **PWA 안됨**: Service Worker 등록 상태 확인

### **로그 확인**
- **백엔드**: 콘솔 로그 확인
- **프론트엔드**: 브라우저 DevTools > Console
- **네트워크**: DevTools > Network 탭
