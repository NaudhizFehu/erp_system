# ERP 시스템 - 직원 관리 화면 개선 작업

## 📋 작업 내용

### 1. 내보내기/가져오기 기능 구현
- 현재 버튼만 존재하고 실제 기능이 동작하지 않음
- 엑셀/CSV 형식으로 직원 데이터 내보내기
- 엑셀/CSV 파일에서 직원 데이터 가져오기

### 2. UI 일관성 개선
- 내보내기/가져오기/직원등록 버튼 크기 통일 필요
- 직원등록창의 input 박스 높이값 통일
- 전체적인 UI 일관성 유지

### 3. 검색 기능 버그 수정
- 직원 관리 화면에서 재직상태를 드롭박스로 변경할 경우
- 하단 리스트가 업데이트되지 않는 오류 수정

## 🔧 기술 스택
- 프론트엔드: TypeScript, React 18, Vite, Tailwind CSS, Shadcn UI
- 백엔드: Java 17, Spring Boot 3
