# ERP 시스템 - High Priority 작업

## High Priority 작업 (필수)

### 1. 영업관리 모듈 프론트엔드 구현
**상태**: 백엔드 API 완료, 프론트엔드 미완료

작업 내용:
1. 고객 관리 화면 - CRUD 기능, 검색 및 필터링, 목록/상세 화면
2. 주문 관리 화면 - 주문 생성/수정/조회, 주문 상태 변경, 주문 내역 관리
3. 견적 관리 화면 - 견적서 생성 및 관리, 견적 승인 프로세스
4. 영업 통계 대시보드 - 실시간 차트, 성과 지표

**예상 기간**: 2주

### 2. 재고관리 모듈 완성
**상태**: 백엔드 일부만 구현

작업 내용:
1. 재고 추적 시스템 - 실시간 재고 현황, 재고 이동 이력
2. 입출고 관리 시스템 - 입고/출고 처리, 재고 이동 기록
3. 상품 관리 프론트엔드 화면 - 상품 CRUD, 카테고리 관리, 가격 관리
4. 재고 대시보드 - 재고 현황 시각화, 재고 경고 알림

**예상 기간**: 2주

- 프론트엔드: TypeScript, React 18, Vite, Tailwind CSS, Shadcn UI
