🚀 ERP 시스템 API 소개

이 API는 현대적인 기업 자원 관리(ERP) 시스템을 위한 완전한 RESTful API입니다. 인사관리, 영업관리, 재고관리, 회계관리, 대시보드 등 기업 운영에 필요한 모든 기능을 제공합니다.

👥

인사관리

직원, 급여, 근태 관리

💼

영업관리

고객, 주문, 견적, 계약 관리

📦

재고관리

상품, 재고, 입출고 관리

💰

회계관리

계정과목, 거래, 예산 관리

📈

대시보드

통계 및 현황 조회

🔐

인증/보안

JWT 기반 인증 시스템

📖 API 문서 보기 (Swagger UI) 📄 OpenAPI JSON

🔧 주요 기능

  • 완전한 CRUD 작업: 모든 엔티티에 대한 생성, 조회, 수정, 삭제
  • 고급 검색 및 필터링: 복잡한 검색 조건과 정렬 기능
  • 페이징 지원: 대용량 데이터 효율적 처리
  • 역할 기반 접근 제어: USER, MANAGER, ADMIN 권한 관리
  • 실시간 통계: 대시보드를 통한 실시간 현황 조회
  • 파일 업로드/다운로드: 문서 및 이미지 관리
  • 감사 로그: 모든 데이터 변경 추적

🔐 인증 방법

모든 API는 JWT Bearer 토큰 인증이 필요합니다.

1단계: 로그인

POST /api/auth/login { "usernameOrEmail": "admin", "password": "password123" }

2단계: 토큰 사용

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

테스트 계정:

  • 관리자: admin / admin123
  • 매니저: manager / manager123
  • 사용자: user / user123
